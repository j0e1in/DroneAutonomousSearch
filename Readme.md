#Drone Search

##Requirememts
###Windows
- node v0.12.x
- cmake-js < 2.0.0 (~1.9.0)
- Nvidia Driver
- CUDA 7.0
- ZED v0.9.1
- Boost library
	- BOOST_INCLUDE_DIRS: ${BOOST_ROOT}
	- BOOST_LIBRARY_DIR_64: ${BOOST_ROOT}\lib64-msvc-12.0


##Build & Run
> $ cmake-js rebuild (to build)

> $ npm run (to run)

> $ npm test (to test)
